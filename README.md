# سیستم تیکتینگ (Help Desk)

پروژه‌ی دانشگاهی سیستم پشتیبانی با PHP و MySQL

## پیش‌نیازها
- PHP 7.4 یا بالاتر
- MySQL 5.7 یا بالاتر
- Apache mod_rewrite فعال

## نصب
1. پایگاه داده `ticketing` را با اجرای فایل `database.sql` ایجاد کنید.
2. اطلاعات اتصال را در `config/database.php` تنظیم کنید.
3. پوشه‌ی `public` را به عنوان Document Root وب سرور تنظیم کنید.
4. دسترسی نوشتن به پوشه `uploads/` بدهید.
5. Cron Job را مطابق توضیحات زیر تنظیم کنید.

## کرون جاب
فایل `cron/close_stale_tickets.php` تیکت‌هایی که ۴۸ ساعت در وضعیت "در انتظار پاسخ" مانده‌اند را می‌بندد.
